Je suis Développeur Web

Aujourd'hui et demain, la dématérialisation va s'accentuer. Les mails ont révolutionné le courrier, la prise de rendez-vous en ligne permet à un entrepreneur d'être moins sollicité par le téléphone, je propose donc à mes clients un accompagnement vers la digitalisation de leur activité. Passionné de longue date par le numérique, je fabrique un site internet comme un artisan fabrique un meuble.

A propos

Après une formation initiale dans la qualité industrielle, j'ai travaillé 10 ans dans plusieurs grosses entreprises de la région Auvergne. Pendant ce temps, j'étais en admiration devant les évolutions technologiques de l'internet. Je me suis auto-formé à l'environnement Web 2.0. Ensuite, j'ai décroché plusieurs formations dans des écoles réputées. Aujourd'hui, je démarre une carrière de Développeur Web freelance, avec pour principal objectif : la satisfaction du client.

Formations web

- Développpeur Web OpenClassRoom (2019)

- Métiers du numérique Simplon (2020)

- Développeur Web Full-Stack (option Design) Studi (2021)

Compétences

Web design, UI / UX

Création de l'identité numérique, logo, charte graphique, expérience de l'utilisateur, confort de l'interface

Sites vitrine / vente en ligne

Conception de sites vitrine, e-commerce, événementiel, avec animations et navigation intuitive. Ajout d'une interface d'administration pour gérer les clients et les produits ou services.

Applications / fonctionnalités spécifiques

Conception / collaboration de projets sur mesure nécessitant l'exploitation de données et la programmation par algorithme.

Bonus

Gestion de projet selon les méthodes agile, versionning selon workflow, administration de serveurs web sous linux pour l'hébergement.
